Introduction

Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured, object-oriented and functional programming.

Steps on getting started with Python:

  1. Open Python.org
  2. Select the right Python verison
  3. Download Python IDLE onto your computer
  4. Open Python
  5. Done!
Data Types

Python currently supports a variety of different data types, which are essential for storing and manipulating different kinds of information. Here are some fundamental Python data types:

  • Integers (int)

    • Used for whole numbers without any decimal points.

    Example: x = 5

  • Floats (float)

    • Used for numbers with decimal points.

    Example: x = 3.14

  • Strings (str)

    • Used for sequences of characters, enclosed in single (' '), double (" "), or triple (''' ' ''') quotes

    Example = "Hello World!"

  • Booleans (bool)

    • Used for representing truth values, either True or False.

    Example = 'is_python_fun = True'

Numbers

Python interpreter functions like a basic calculator, allowing you to input expressions, and it will output the computed values. The syntax for expressions is simple: you can use the standard arithmetic operators (+, -, *, /) for mathematical operations, and parentheses (()) for grouping.

For example:

print(2 + 2)

print(10 / 2)

print(5 * 10 + 2)

If Statements

the if statement is a fundamental control structure that allows you to conditionally execute a block of code based on a specified condition. Here's a brief explanation:

        
        x = 10
        if x > 5:
            print("x is greater than 5")
        
      
        
        y = 3
        if y % 2 == 0:
            print("y is even")
        else:
            print("y is odd")
        
      
        
        z = 0
        if z > 0:
            print("z is positive")
        elif z < 0:
            print("z is negative")
        else:
            print("z is zero")
While Statements

A while statement is used to create a loop that continues executing as long as a specified condition is true. The general syntax of a while loop in Python is as follows:

        
          # Example: Counting down from 5 to 1
          count = 5

          while count > 0:
              print(count)
              count -= 1

          print("Blast off!")
